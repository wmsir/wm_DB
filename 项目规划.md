# 一、项目定位

wmDB（Perfect Database）定位为 **企业级数据库治理平台（Database Governance Platform）**，以 SQL 审核为核心能力，逐步演进为覆盖数据库全生命周期治理的商业产品。

平台不仅提供 SQL 审核，还将覆盖：

* 数据库变更管理
* 数据库安全治理
* DevOps 数据库发布
* 数据资产管理
* 数据审计
* AI 数据库助手
* 数据库智能运维（AIOps）

最终打造国产数据库治理平台。
